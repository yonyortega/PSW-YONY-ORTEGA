Principales Etiquetas HTML

Hay una serie de etiquetas que son las más usadas para crear cualquier documento HTML, a continuación las explico

1-Body


Es para el contendido de la pagina

2-Head


Informacion sobre la pagina

3-Div


Sirve para realizar division del contenido

4-a

Para llamar enlaces

5-Strong


Sirve para poner el texto en negrita

6-br


Para saltos de linea

7-H1..H6


sirven para realizar diferentes titulos

8-Img


para insertar imagenes

9-ol,ul,li


para listas ordenadas, desordenadas y para elementos dentro de la lista

10-tr,td

br para filas y columnas dentro de la tabla
